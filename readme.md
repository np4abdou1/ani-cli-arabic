## ðŸ“– About
This project is a command-line interface (CLI) tool designed to let you search, select, and stream anime directly from your terminal. 

It is heavily inspired by the legendary [pystardust/ani-cli](https://github.com/pystardust/ani-cli), but with a specific focus for the Arab community: **Native Arabic Subtitles.**

### âœ¨ Features
* **Terminal Interface:** Lightweight and fast. No browser required.
* **Arabic Subtitles:** Fetches anime sources specifically with Arabic translation.
* **Search & Stream:** Easy-to-use menu system to find your favorite shows.
